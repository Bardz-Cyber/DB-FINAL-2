# UI Design Notes

The C# WinForms interface uses a modern dark dashboard style.

## Main Screens
- Dashboard cards for Total Products, Low Stock Items, and Inventory Value
- Search bar for product name, SKU, or category
- Product table with category, supplier, price, quantity, and stock status
- Product form for adding/editing inventory items
- Stock transaction form for stock-in and stock-out
- Transaction report window

## Color Palette
- Background: #0F172A
- Cards/Panels: #1E293B
- Primary Accent: #0284C7 / #38BDF8
- Text: #E2E8F0
- Warning/Danger: #F87171
