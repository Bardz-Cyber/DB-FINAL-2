Inventory Management System - C# WinForms
Module 12 Final Database Project

Contents:
1. src/InventorySystemCS - C# WinForms desktop source code
2. sql - SQLite schema, sample data, and query report scripts
3. docs - project report, presentation script, and UI notes
4. presentation - PowerPoint slide deck
5. assets - ER diagram and UI mockup images

How to Run:
1. Install Visual Studio 2022 or later with .NET desktop development workload.
2. Open src/InventorySystemCS/InventorySystemCS.csproj.
3. Restore NuGet packages when prompted.
4. Build and run the project.
5. The app automatically creates inventory.db on first launch.

Main Features:
- Product CRUD
- Category and supplier relational data
- Stock-in and stock-out transactions
- Low stock detection
- Dashboard totals
- Transaction report
- Clean dark modern WinForms UI

Database:
The C# app uses SQLite through Microsoft.Data.Sqlite.
The SQL scripts are included for project checking and documentation.
